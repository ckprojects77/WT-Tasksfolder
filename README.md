# Web Technologies Lab

This repository contains my Web Technologies Lab assignments completed as part of my **B.Tech Information Technology** course at **Anil Neerukonda Institute of Technology & Sciences**. Each assignment is organized in a separate folder and demonstrates the concepts learned during the lab sessions.

---

## Tasks

### Task 1 – Portfolio Website
Developed a responsive personal portfolio website using **HTML5** and **CSS3** with sections including Home, About, Skills, Projects, and Contact.

### Task 2 – Student Profile Webpage
Built a responsive Student Profile webpage using **HTML5** and **Bootstrap 5**, featuring a Navbar, Profile Card, Academic Details, Skills, Contact, and Footer.

### Task 3 – Student Registration Form with Validation
Designed a responsive student registration form using **HTML**, **CSS**, and **JavaScript** with client-side validation and error handling.

### Task 4 – Personal Portfolio Website with Contact Form Validation
Created a responsive personal portfolio website using **HTML**, **CSS**, and **JavaScript** with a validated contact form and interactive user experience.
