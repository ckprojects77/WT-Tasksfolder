# Web Technologies Lab

This repository contains my Web Technologies Lab assignments completed as part of my B.Tech Information Technology course. Each assignment is organized in a separate folder and demonstrates the concepts learned during the lab sessions.

## Tasks

### Task 1 – Portfolio Website
Developed a personal portfolio website using HTML5 and CSS3. The webpage includes sections such as About, Skills, Projects, and Contact with a responsive and visually appealing design.

### Task 2 – Student Profile Webpage
Created a responsive Student Profile webpage using HTML5 and Bootstrap 5. The webpage features a Bootstrap Navbar, Profile Card, Academic Details Table, Skills Section, Contact Section, and Footer while utilizing Bootstrap's Grid System for responsiveness.

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
